# Startup Ideas My Friend Shot Down

A scrollytelling comedy webpage: ten parody startup logos, each a pun on a real brand, born from a text thread about prediction markets and Silicon Valley.

**▶ Live site:** https://no-macaroon1670.github.io/StartupLogos/

Scroll through a flat, deadpan description of each "idea" — then the logo lands the joke, with a one-line slogan beneath.

## The lineup

Metaculus · The Bettor's Business Bureau · BestBuy · PrinceTon · Universe of Chicago · Proctor: Gamble · metacritics · O'Pen Al · An thot pic · Na'viDIA

## How it's built

A single self-contained `index.html` — hand-coded inline-SVG logos and scroll-driven reveals (Intersection-style, via `getBoundingClientRect`). No build step, no dependencies. Open the file directly or visit the live site and scroll.

All logos are parody/satire of existing brands.
